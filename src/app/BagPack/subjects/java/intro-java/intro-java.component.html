    • Object Oriented Programming

    Advantages / Features :-

  • Focus on data rather than functions
  • Follow bottom-up approach
  • Program divided into objects
  • Objects can communicate through functions
  • Any data and function in object can easily added
  • Objects can be reusable
  • Data is kept hidden from external objects

    • Application areas of OOP :-

  • User interface design
  • Real time System
  • Simulation and Modeling
  • Object Oriented Databases
  • Artificial Intelligence
  • Games, Entertainment and Multi-media Software

    • Concepts / Terminologies in OOPs :-

    1] Class

  • A class is user define data type , Which consist of data and methods of an object
  • It is also called as blueprint of an object
  • A class is a group of objects which have common properties.


    • 2] Object

  • Object is a instance (result) of a class






    • 3] Data Abstraction

  • Data abstraction is a process of hiding certain details and showing only essential information to the user


    • 4] Data Encapsulation

  • Data Encapsulation is the process by which data and the objects (code that acts upon them ) or methods are integrated as a single unit


    • 5] Inheritance

  • Inheritance is a mechanism where in a new class is derived from an existing class


    • 6] Polymorphism

  • Polymorphism is a concept by which we can perform single action in different way
  • In below img Sound() fuction is used in different ways


    • 7] Message-Passing

  • Objects are communicate with one another by sending and receiving information to each other
