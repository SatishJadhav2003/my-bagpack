Intro to Data Science :-

  • Data science is an interdisciplinary field that uses scientific methods, processes, algorithms and systems to extract or extrapolate knowledge and insights from noisy, structured and unstructured data,[1][2] and apply knowledge from data across a broad range of application domains



  • Data Science and Associated Fields


  • Process of Data Science

    • Advantages of Data Science :-

  • It helps to extract meaningfull information from raw data
  • It is widely used in health-care, banking, consultancy services, and e-commerce industries. Data Science is a very versatile field
  • It improve a quality of data
  • It improves a quality of services and Products

    • Disadvantages of Data Science :-

  • Problem of Data Privacy
  • Large Amount of Domain Knowledge Required
  • The tools used for data science are more expensive to use to obtain information
  • Difficult to selection of Tools

    • The 3 V's of Data Science :-

  • The 3 V's of data science are : 1) Volume, 2) Velocity, 3) Variety
  • Volume => size and scope of data
  • Velocity => A speed at which data is accumulated
  • Variety => Different type of data

    • Data sources :-

    1) Open data sources

  • Open data is that some data should be freely available in a public domain that can be used by anyone as they wish, without any restrictions

    • 2) Social media data

  • Social media data provides commonly available information on social media channels published by social media users.
  • We can get social media data from different social media data API (Application programming interface)

    • 3) Multi-Modal data

  • Multimodal data refers to data that spans different types and contexts (e.g., imaging, text, or genetics)

    • 3) Standard Datasets

  • A datasets or data set is simply a collection of data
  • Most common format for datasets is a CSV format
